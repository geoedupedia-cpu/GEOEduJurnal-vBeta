Tentang Aplikasi

GeoEdu Jurnal Pembelajaran dibuat untuk membantu guru mencatat kegiatan pembelajaran harian secara rapi dan mudah direkap. Data kini tersimpan otomatis di perangkat ini (IndexedDB) sehingga tetap ada saat aplikasi dibuka kembali, tanpa perlu impor manual. Tombol Ekspor Data (JSON) di halaman Laporan Jurnal tetap tersedia untuk mencadangkan atau memindahkan data ke perangkat lain, dan dapat dimuat kembali kapan saja melalui tombol Impor Data (JSON).

Untuk pertanyaan, masukan, atau kerja sama seputar GEOEduPedia, silakan hubungi surel di atas.

Log Penggunaan & Perubahan Data

Rekam jejak aktivitas aplikasi ini (dibuka, disimpan, diubah, dihapus, ekspor/impor). Tersimpan otomatis di perangkat ini, maksimal 1000 catatan terakhir. Halaman ini hanya menampilkan 5 aktivitas terbaru; gunakan tombol di bawah untuk mengunduh seluruh riwayat log.
